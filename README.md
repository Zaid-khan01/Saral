# 🌍 About Saral

**Saral** is a futuristic smart queue management platform designed to digitize public waiting systems and eliminate long physical queues.

The platform allows users to:

- 🎟️ Book digital queue tokens
- 📡 Track queue movement in real time
- 🔔 Get smart notifications before their turn
- 🏥 Access multiple nearby organizations instantly

Saral aims to modernize queue systems for:

- Hospitals
- Government Offices
- Ration Shops
- Service Centers
- Public Utility Centers

# ⚡ Installation & Setup

# 🔹 Clone Repository

```bash
git clone https://github.com/your-username/saral.git

cd saral
```

---

# 🔹 Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# 🔹 Backend Setup

```bash
cd backend

npm install
```

---

# 🔹 Create Environment Variables

Create a `.env` file inside backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# 🔹 Start Backend Server

```bash
npm start
```
